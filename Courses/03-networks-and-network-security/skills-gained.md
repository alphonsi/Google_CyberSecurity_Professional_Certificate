# Skills Gained - Networks and Network Security (Google Cybersecurity Professional Certificate)

This document outlines the skills acquired through the **Networks and Network Security** course, which is the third course in the Google Cybersecurity Professional Certificate program on Coursera.

## Technical Skills

- Network fundamentals and concepts
- OSI model and TCP/IP model
- Network layers and their functions
- TCP and UDP protocols
- IP addressing (IPv4 and IPv6)
- DNS and DHCP
- HTTP/HTTPS and other application layer protocols
- Network devices (routers, switches, firewalls, IDS/IPS)
- Access control lists (ACLs)
- Virtual private networks (VPNs)
- Network segmentation
- Network monitoring and analysis
- Packet sniffing and analysis (Wireshark)
- Wireless network security
- Wi-Fi security protocols (WEP, WPA, WPA2, WPA3)
- Network hardening techniques
- Network security policies and procedures
- Network incident response
- Network performance optimization
