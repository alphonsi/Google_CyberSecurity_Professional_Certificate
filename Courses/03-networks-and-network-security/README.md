## Overview

The **Networks and Network Security** course is the third step in the Google Cybersecurity Professional Certificate program. It teaches learners about network fundamentals, network security concepts, and practical skills for securing networks. This course covers topics such as network protocols, network devices, network security controls, and network monitoring.

## Course Structure

The course follows the Google Cybersecurity Professional Certificate curriculum, divided into the following modules:

1. **Introduction to Networks**
2. **Network Fundamentals**
3. **Network Protocols**
4. **Network Devices**
5. **Network Security Controls**
6. **Network Monitoring and Analysis**
7. **Wireless Network Security**
8. **Network Hardening and Optimization**

## Key Topics

### Introduction to Networks
- What is a network and how it works
- Types of networks (LAN, WAN, MAN, PAN)
- Network topologies (star, mesh, bus, ring)
- Network addressing and routing

### Network Fundamentals
- OSI model and TCP/IP model
- Network layers and their functions
- Network communication basics
- Network performance and reliability

### Network Protocols
- TCP and UDP protocols
- IP addressing (IPv4 and IPv6)
- DNS and DHCP
- HTTP/HTTPS and other application layer protocols

### Network Devices
- Routers and switches
- Firewalls and intrusion detection/prevention systems
- Load balancers and proxies
- Wireless access points and controllers

### Network Security Controls
- Access control lists (ACLs)
- Virtual private networks (VPNs)
- Network segmentation
- Intrusion detection and prevention systems (IDS/IPS)

### Network Monitoring and Analysis
- Network monitoring tools and techniques
- Packet sniffing and analysis
- Network performance monitoring
- Security event correlation

### Wireless Network Security
- Wireless network fundamentals
- Wi-Fi security protocols (WEP, WPA, WPA2, WPA3)
- Wireless network threats and vulnerabilities
- Wireless network security best practices

### Network Hardening and Optimization
- Network hardening techniques
- Network performance optimization
- Network security policies and procedures
- Network incident response
