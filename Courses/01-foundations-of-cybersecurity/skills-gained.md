# Skills Gained - Foundations of Cybersecurity (Google Cybersecurity Professional Certificate)

This document outlines the skills acquired through the **Foundations of Cybersecurity** course, which is the first course in the Google Cybersecurity Professional Certificate program on Coursera.

- Cybersecurity terminology and concepts
- Core security principles (CIA Triad, Defense in Depth, Least Privilege)
- Threat analysis and identification
- Vulnerability recognition and assessment
- Risk assessment using industry frameworks
- NIST Cybersecurity Framework (CSF)
- NIST Risk Management Framework (RMF)
- Regulatory compliance (GDPR, HIPAA, PCI DSS, CCPA)
- Security operations basics
- Security monitoring processes
- Incident detection fundamentals
- Threat intelligence utilization
- Security controls implementation
- Security documentation creation
- Security information and event management (SIEM) basics
- Vulnerability management processes
- Incident response basics
