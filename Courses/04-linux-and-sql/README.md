## Overview

The **Linux and SQL** course is the fourth step in the Google Cybersecurity Professional Certificate program. It teaches learners Linux operating system fundamentals and SQL database skills, which are essential for cybersecurity professionals. This course covers topics such as Linux commands, file system management, user management, SQL queries, and database security.

## Course Structure

The course follows the Google Cybersecurity Professional Certificate curriculum, divided into the following modules:

1. **Introduction to Linux**
2. **Linux File System**
3. **Linux Commands and Shell Scripting**
4. **Linux User and Group Management**
5. **Introduction to SQL**
6. **SQL Queries**
7. **SQL Joins and Subqueries**
8. **Database Security**

## Key Topics

### Introduction to Linux
- What is Linux and why it's important for cybersecurity
- Linux distributions (Ubuntu, CentOS, Debian)
- Linux file system hierarchy
- Basic Linux commands

### Linux File System
- File system structure (/, /home, /etc, /var, /tmp)
- File permissions and ownership
- File types (regular files, directories, symbolic links)
- File manipulation commands (cp, mv, rm, mkdir, rmdir)

### Linux Commands and Shell Scripting
- Basic Linux commands (ls, cd, pwd, cat, grep)
- Advanced Linux commands (find, grep, awk, sed)
- Shell scripting basics
- Creating and running shell scripts

### Linux User and Group Management
- User accounts and passwords
- Group management
- Permissions and access control
- User and group commands (useradd, usermod, userdel, groupadd, groupmod, groupdel)

### Introduction to SQL
- What is SQL and why it's important for cybersecurity
- Relational databases and tables
- SQL data types
- Basic SQL statements

### SQL Queries
- SELECT statement
- WHERE clause
- ORDER BY clause
- LIMIT clause
- Aggregate functions (COUNT, SUM, AVG, MIN, MAX)

### SQL Joins and Subqueries
- INNER JOIN
- LEFT JOIN and RIGHT JOIN
- FULL OUTER JOIN
- Subqueries

### Database Security
- Database authentication and authorization
- User accounts and permissions
- Database encryption
- Database auditing and monitoring
