# Skills Gained - Linux and SQL (Google Cybersecurity Professional Certificate)

This document outlines the skills acquired through the **Linux and SQL** course, which is the fourth course in the Google Cybersecurity Professional Certificate program on Coursera.

- Shell scripting basics
- Linux user and group management
- SQL database fundamentals
- SQL queries and statements
- SELECT statement with WHERE, ORDER BY, and LIMIT clauses
- Aggregate functions (COUNT, SUM, AVG, MIN, MAX)
- SQL joins (INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN)
- Subqueries
- Database security
- Database authentication and authorization
- User accounts and permissions
- Database encryption
- Database auditing and monitoring
